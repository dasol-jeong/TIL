# git clone으로 하는 법

## git clone하는 순서

### 1. github에 new-repo만들기

### 2. git bash에서 git clone {repo path} 넣어서 clone

### 3. mkdir로 디렉터리  생성

### 4. touch로 파일 생성

### 5. git add 생성한 파일 해주고

### 6. git commit 해서 제목과 내용쓰기

### 7. git push origin main 해서 깃헙 레파지토리에 push 해주기
